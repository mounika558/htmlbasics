Website vs Web Application

Website

Website is a collection of web pages and its is structured using HTML css and js. This
is like a static page where the contents of the page don't change dynamically .
for example a portfolio websites , business websites. search engine is used to find the webpages from the websites.

  • Advantage
  • Easy to use
  • fast in loading
  • DISAdvantage
  • fraudulent website
  • invasive advertising
Web Application

Web Application is a software which run over a browser and it is a dynamic one
which changes according to the users need for example e-commerce apps is a web apps (myntra, Amazon etc)
It has rendering methods like client side rendering and server side rendering .
We should consider both and work on it to obtain a efficient solution.

  • ADVANTAGE
  • cost effective
  • Run easily
  • up to date
  • DISADVANTAGE
  • Reduced speed
  • less secure
  • internet reliance
Elements

Self closing tags : (img. br, hr, input etc..) these tags don't have closing tags .
These tags are called as empty tags as there

is no content to display and those tags can't have child tags
